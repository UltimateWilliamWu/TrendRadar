投资相关性：高。

【市场主线1】
美国利率路径再定价仍是今天全球资产的第一驱动，核心不是“是否降息”，而是美联储新主席沃什上台后，市场开始重新定价更偏鹰派、更强调通胀约束的政策框架。传导链条：美联储人事变化/沃勒鹰派表态 -> 降息预期后移、甚至年底加息概率被抬升 -> 美债收益率上行、美元偏强、成长股估值承压、黄金与高久期资产波动加大 -> A股/港股的外资定价、科技成长和高估值板块承压。

【资产映射】
美债、美元、黄金、全球成长股、港股科技、A股成长风格。

【证据强度：中】
（人事与表态来自热榜/媒体，政策路径尚未由官方会议纪要或数据验证）。

【市场主线2】
中美科技股内部出现“AI资本开支继续扩张，但估值分化加剧”的结构性行情。美股端科技股力撑指数、芯片与量子概念走强，但英伟达当周回撤，存储芯片和部分AI基础设施股更强；A股/港股端则出现半导体ETF、算力巨头超聚变IPO、AI基础设施与产业知识产权中心等信号。传导链条：AI算力/基础设施资本开支维持高位 -> 订单与供给链景气支撑硬件、存储、服务器、光模块、数据中心电力链 -> 但龙头估值对利率更敏感，导致“业绩兑现强于叙事”的分化交易 -> 资金更偏向有订单、现金流和回购的标的。

【资产映射】
美股芯片、存储、AI基础设施；A股半导体、算力、服务器、数据中心电力；港股互联网与AI应用。

【证据强度：中】
（RSS有公司与产业活动增量，但缺少统一的行业订单数据）。

【市场主线3】
中国监管对跨境证券经营与非法展业的集中整治，直接影响中概券商与跨境互联网券商估值，同时强化境内资本市场“合规优先”的政策信号。传导链条：证监会等八部门整治非法跨境证券期货基金经营活动 + 对富途、老虎、长桥拟严厉处罚 -> 跨境展业模式的监管风险上升、获客与业务扩张受限 -> 中概券商/跨境金融平台估值承压，境内券商合规与渠道优势相对受益 -> 港股中概金融与美股中概券商波动加大。

【资产映射】
富途、老虎证券、长桥、港股券商、A股券商、跨境金融相关标的。

【证据强度：强】
（来自监管与媒体披露，且多源一致）。
舆论风向争议
【预期差1】
市场对美联储“新主席=更快降息”的幻想正在被快速修正，但热榜叙事仍在放大“变天”本身，真正影响定价的是利率路径是否更久更高。证据更强的一侧是沃勒鹰派措辞、市场对年底加息的重新计价；证据较弱的一侧是对新主席个人风格的过度解读。可能的过度交易：短线过度押注美元走强、长久期成长继续杀估值；但若后续数据并未恶化，利率上行幅度可能有限。

【证据强度：中】
。
