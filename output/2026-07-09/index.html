- 传导链条：就业仍稳 -> 降息紧迫性下降；但短端资金利差转正/买盘激增 -> 市场对流动性结构、回购与资金面更敏感；纪要偏鹰 -> 长端利率下行空间受限 -> 成长股估值弹性受约束，但若AI资本开支继续强，科技股仍可相对跑赢。
- 主要影响资产/市场：美债收益率曲线、美元、黄金、纳指/成长股、信用利差、波动率。
- 证据强度：强（EIA/劳工数据/美联储纪要/纽约联储口径均为硬信息）。

【市场主线3：原油/天然气与地缘扰动带来能源价格分化，天然气偏弱、油价受供给与监管事件扰动】

- 传导链条：天然气库存高于预期+出口设施检修 -> 现货/期货承压 -> 美国气价、LNG链、化工成本边际下行；原油交易机制与地缘扰动 -> 油价波动率抬升 -> 能源股、航运、化工、通胀预期受影响。
- 主要影响资产/市场：WTI/Brent、天然气、能源股、化工、通胀预期、风险偏好。
- 证据强度：中到强（EIA数据强，地缘与“市场消息”部分证据强度中等）。

- 过热信号：热榜集中在“AI热情”“中国AI产业链”“苹果端侧大模型”“OpenAI效率提升”“Meta自研芯片”“AI基建高景气”。

- 1周：若美联储纪要偏鹰被市场消化，而AI资本开支与端侧AI消息继续密集，成长风格可能维持相对收益；反之若SOFR/FF基差异常持续，说明资金面压力会压制高估值板块。
- 1月：指数层面更像“AI驱动的结构性牛市/修复”，但不是全面风险偏好扩张；若国内CPI/PPI仍温和、政策继续偏托底，A股成长与科技的相对优势更明显。

- 日内最热主题1：AI算力/半导体。代表股：英伟达、AMD、台积电、博通、应用材料；A股/港股关注中芯国际、寒武纪、海光信息、北方华创、中微公司、沪电股份、工业富联、澜起科技。证据强度：强。
- 日内最热主题2：存储与HBM/DRAM。代表股：SK海力士、美光、三星电子；A股/港股关注兆易创新、江波龙、佰维存储、德明利、香农芯创。证据强度：强。
- 日内最热主题3：端侧AI与手机/消费电子。代表股：苹果、Meta、OpenAI生态相关；A股关注立讯精密、歌尔股份、蓝思科技、传音控股、京东方A。证据强度：中。
