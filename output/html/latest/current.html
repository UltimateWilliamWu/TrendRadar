核心热点态势
投资相关性：高。

【市场主线1】

【：美国通胀继续降温，利率路径偏鸽，支撑美股风险偏好但对存储/半导体形成结构分化】

【证据强度：强】
- 关键事实：美国6月PPI同比5.5%、环比创六年来最大跌幅，核心PPI同比4.7%低于预期；市场同时解读为通胀压力缓和。
- 传导链条：PPI降温 -> 美联储短端加息/高利率预期边际回落 -> 美债收益率与美元偏弱/波动率下降 -> 科技成长股估值修复 -> 美股指数受大型科技支撑走强。
- 资产映射：美债偏多、美元偏弱、黄金偏中性偏多、纳指/科技股偏多；但存储芯片、部分半导体设备因周期与个股事件承压。

【市场主线2：半导体与AI链条出现“指数强、细分弱”的分化，资金从存储/设备撤出，转向AI应用、云与平台型龙头】

【证据强度：中】
- 关键事实：A股创业板、科创50回调，半导体、算力硬件持续调整，存储龙头再跌停；港股恒科指大涨、科网股全线走强；美股中概股普涨但美光、闪迪暴跌。
- 传导链条：AI叙事仍强 -> 但市场开始区分“AI需求确定性”与“存储/设备短期价格与监管扰动” -> 资金从高拥挤硬件链切向应用、平台、互联网与部分软件服务 -> 指数层面风格切换，A股偏防御/消费/医药，港股互联网相对占优。
- 资产映射：A股半导体/算力硬件偏空，港股互联网偏多，美股大型科技偏多但存储链偏空。

【市场主线3：全球贸易与地缘摩擦升温，商品与风险资产受扰动，能源和黄金的尾部风险溢价上升】

【证据强度：中】
- 关键事实：美国对巴西部分商品加征25%关税并启动301条款行动；巴西拟通过WTO挑战；同时伊朗冲突持续、美国继续打击伊朗，战争成本预期抬升；布油四连涨站上85美元。
- 传导链条：关税/地缘冲突 -> 供应链与运输成本上升、能源风险溢价抬升 -> 原油、黄金、军工与部分资源品受益 -> 风险偏好受压，全球股市风格更偏防御。
- 资产映射：原油偏多、黄金偏多、航运/化工/航空成本端承压；若冲突升级，信用利差与波动率可能抬升。
舆论风向争议
【预期差1：市场对“通胀已彻底回落”的定价可能过快，证据更强的一侧仍是“通胀下行但粘性未消失”】

【证据强度：强】
- 热榜偏向把PPI降温解读为全面利好，但RSS与宏观评论同时提示：美国褐皮书、韩国央行表态都在强调AI、能源与汇率仍可能推高价格粘性。
- 结论：短线交易上，美债和科技股受益更确定；但“降息交易一把梭”证据不足，需观察CPI、就业和服务通胀。

【预期差2：AI叙事并未退潮，但资金拥挤度已明显上升，硬件链条出现去泡沫式分化】

【证据强度：中】
- 热榜大量围绕WAIC、机器人、英伟达合作、AI办公室展开，情绪很热；但A股半导体/存储/设备同步回调，说明“AI=全面买入硬件”已不成立。
- 更强证据来自价格行为：存储龙头跌停、澜起科技大跌、半导体设备ETF走弱，说明市场在重新定价个股与子行业风险，而不是单纯追逐概念。
- 结论：AI应用、平台、云服务的相对强度可能优于纯硬件；硬件链条内部需要更严格区分订单兑现与估值拥挤。

【预期差3：巴西关税与伊朗冲突的情绪热度高，但对A股直接定价影响有限，更多是商品与全球风险偏好渠道】

【证据强度：中】
- 热榜关注度高，但对中国股票市场的直接映射主要通过原油、黄金、航运、化工成本和风险偏好，而非单一行业基本面。
- 若后续没有进一步升级，相关交易可能快速降温，投资相关性中等偏低。
